# Foreign Workers in Sweden

Data-driven journalism for The Local Sweden: one decade of labor migration seen
through Migrationsverket's own statistics — who gets work permits, in which
occupations, how decisions split between first-time grants and extensions, and how
long recruitment takes.

```
.
├── raw-data/              Migrationsverket annual reports + granted-permit workbooks, and the scripts that mine them
└── migration-statistics/  the master & derived CSVs and the analysis notebook
```

Start with `raw-data/README.md` for sources and provenance, then
`migration-statistics/migration-analysis.ipynb` for the analysis.
